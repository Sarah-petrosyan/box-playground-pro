## About the Project

This project is an interactive **CSS styling playground** where users can experiment with different visual properties and instantly see the results on a preview box.

Users can modify styles such as text color, font size, transformations, transitions, and box shadows using input controls. The application updates the styles dynamically, demonstrating how JavaScript can manipulate CSS properties in real time.

The project focuses on building a **dynamic UI with configurable style controls**, using JavaScript to connect input elements with CSS properties.

This project was built to practice **DOM manipulation, event handling, and dynamic styling with JavaScript**.

✅ Built with **HTML, CSS, and JavaScript**  
✅ Real-time style editing  
✅ Interactive transform and shadow controls  
✅ Config-driven styling system  
✅ Beginner-friendly JavaScript project
